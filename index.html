Here you can find a version of papaya CMS 5.5.x with additional patches.

Features

  • supports sub themes
  • improved image gallery module
  • action dispatcher support on delete pages
  • added support for dynamic surfer data groups in community profile
  • better support of dynamic surfer data groups in outputs
  • improved outputs in default-theme, e.g. dialogs
  • and more changes -> http://goo.gl/54cfe
