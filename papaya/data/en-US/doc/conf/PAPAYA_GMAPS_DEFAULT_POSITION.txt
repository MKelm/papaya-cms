Default position, consisting of two decimal values separated by a comma. These two values define the geographical latitude and longitude, respectively. The dot is used as decimal separator. The first value represents the latitude, the second one the longitude:<ul><li>Range of values for the latitude: -90 to 90<br /></li><li>Range of values for the longitude: -180 to 180<br /></li><li>Each value can have an arbitrary number of decimal places.<br /></li></ul>
