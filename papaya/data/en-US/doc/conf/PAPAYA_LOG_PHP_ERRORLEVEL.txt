Activate output of PHP error messages and set the error message level. The error reporting levels are used to specify wich errors to report. You can choose between the following levels and combinations of levels:<ul><li><tt>E_ALL ^ E_NOTICE</tt>: Issues warnings and error messages, but ignores notices.<br /></li><li><tt>E_ALL ^ (E_NOTICE | E_USER_NOTICE)</tt>: Issues warnings and error messages, ignores user defined and runtime specific notices.<br /></li><li><tt>E_ALL</tt>: Issues all types of messages, i.e. errors, warnings, and notices.<br /></li></ul>You can learn more about the error reporting levels in your PHP documentation.
