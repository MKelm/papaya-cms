This score-value expresses the result of the ham-check for a given entry (e.g. an email or a forum entry) as percentage.<br /><br />The value PAPAYA_SPAM_SCOREMIN_PERCENT is used to define the threshold for the ham recognition.<br /><br />For example: If this option was set to a value of 50, papaya CMS would require entries to reach a ham-probability of at least 50% to be accepted.
