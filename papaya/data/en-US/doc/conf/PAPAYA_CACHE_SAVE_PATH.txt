This option allows you to optionally configure the selected cache service. Memcache allows you to set the following options: <code>tcp://hostname:12100?persistent=0</code> In case that memcache is selected as service and this field is left blank, the memcache service will use the option given in the <tt>php.ini</tt> file, <code>session.save_path</code>, but only if memcache has been configured as session handler in the <tt>php.ini</tt>.
