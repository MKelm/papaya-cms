Defined the text used for the alt attribute in image elements for images in the MediaDB:<ul><li><em>explicit only</em>: The <code>alt</code> attribute is used when the alt text is given, otherwise the <code>alt</code> attribute is left out.<br /></li><li><em>description</em>: A shortened version of the description text from the image properties is used for the <code>alt</code> attribute.<br /></li></ul>
