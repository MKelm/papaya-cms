Configuration of the search options for database fields of type TEXT. Choices to select from the drop down list:<ul><li><em>generic SQL (Like)</em>: Performs a simple search. The function "LIKE" allows only to search one field.<br /></li><li><em>MySQL FULLTEXT</em>: Performs a simple full text search. You can give multiple fields.<br /></li><li><em>MySQL FULLTEXT BOOLEAN</em>: Performs a full text search in boolean mode (MySQL version 4.1.x and later). You can learn more about the search modes in the manual of your MySQL server.<br /></li></ul>
