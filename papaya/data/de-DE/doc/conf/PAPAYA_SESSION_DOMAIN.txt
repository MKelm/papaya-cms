Domain, unter der das Session-Cookie gesetzt wird. In der Grundeinstellung von <tt>session.cookie_domain</tt> ist keine Domain angegeben. Stattdessen wird immer der Hostname des Servers verwendet. papaya CMS kann jedoch mehrere Subdomains verwalten. Damit Nutzer sich nicht jedes Mal neu anmelden müssen, wenn sie die Subdomain wechseln, können Sie bspw. so etwas wie <code>domain.tld</code> angeben.
